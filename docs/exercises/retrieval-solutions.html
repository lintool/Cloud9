Boolean Retrieval Exercise: Solutions

This page presents solutions for the Boolean retrieval exercise. The main class for performing retrieval is edu.umd.cloud9.example.ir.BooleanRetrieval. Assuming the inverted index has already been constructed and resides on HDFS, the command-line invocation is as follows:

etc/hadoop-cluster.sh edu.umd.cloud9.example.ir.BooleanRetrieval -index index -collection bible+shakes.nopunc

In this case, we're reading indexes directly from HDFS. Alternative, you can copy the indexes local and use the etc/run.sh launch script:

etc/run.sh edu.umd.cloud9.example.ir.BooleanRetrieval -index index -collection bible+shakes.nopunc
